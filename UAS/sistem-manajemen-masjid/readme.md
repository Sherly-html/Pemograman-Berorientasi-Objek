# Aktifkan virtual environment
Windows: 
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

Kalau berhasil, terminal kamu akan menampilkan (venv) di depannya:
- (venv) C:\Users\kamu\sistem-manajemen-masjid>

# Install Dependencies

Setelah venv aktif, jalankan perintah ini satu per satu:

```bash
pip install flask
pip install flask-login
pip install flask-wtf
```
